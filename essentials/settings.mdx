title: 'Github Installation'
description: 'There are 2 ways to install Github on Panaptico, either through the Onboard screen or the Install Application Menu on the left hand side of navigation'
icon: 'gear'
---

To use Github we use an authentication system which generates a token when you grant access to our app.
Access tokens created by apps are automatically authorized for SAML SSO.
Panaptico only uses read only privileges on your github account.
